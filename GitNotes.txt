git init demo : To initialise a new git repository with the name of demo.

git status: to check the status of files in the staging area.

git add: To send a file to the staging area.

git add . : To add all the files in the staging area.

git show/log : Obvious.

git commit -m" Hey there " 

git init . : Git bash here

git commit -am" Modify and commit"

git checkout --(name of file ) : Clear the changes from the area

Some not so good commands : rename , ignoring some files and comparing , rm

git branch: To see which branch are we currently on 

git branch -a : To see all the branches 

git checkout -b (name of file) : create a new branch

git diff (name of new branch) master : check the differences

git checkout master : Return to the master branch 

git merge (name of new branch) : Merging new branch

git checkout -d (name of branch) : Deleting a branch

git remote add orgin (link) : Connecting local repo to github

git push -u origin master --tags : First time pushing
